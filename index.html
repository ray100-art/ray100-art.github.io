Who I Am
About Me

Hey, I'm Brian Ndung'u — a Software Developer from Chuka, Kenya who genuinely loves building things. Not just writing code, but actually solving problems that matter to real people.

I started with the basics — C, algorithms, logic — and kept going. Today I work across the full stack: Java Spring Boot backends, PHP web apps, MySQL databases, clean HTML/CSS/JS frontends, and even desktop applications. I'm also deeply interested in Machine Learning and AI — from rule-based expert systems to modern intelligent applications — and I build mobile apps too.

Right now I'm most proud of ParkNairobi — a smart parking system I built for Kenya with real-time slot tracking, M-Pesa payments, live maps and GPS navigation. It's the kind of project that shows what's possible when you combine good engineering with a real local problem.

When I'm not coding, I'm thinking about what to build next. That drive is what keeps me growing.
